git init
Git 저장소 초기화
git status
Git 상태(새롭게 추가되거나, 수정된 내용) 확인
git add [file, *]
파일 버전관리 스냅샷(Snapshots)
git commit -m "메모 내용"
영구적으로 버전관리 스냅샷 파일 기록
git log
커미트 히스토리 확인
git commit -a
스테이지 단계를 건너 뛰고 커밋하기
git commit --amend
완료한 커미트 덮어쓰기
git checkout -- index.html
개별 파일 복구하기(이전파일로 되돌림)
git remote add origin "주소"
원격 저장소 등록
git push origin master
원격 저장소에 프로젝트 업데이트하기
git diff
워킹디렉토리 단계에서 파일의 변화 확인하기
git diff --staged
스테이지 단게에서 파일의 변화 확인하기
git reset HEAD^
최근 커미트 결과 취소하기 실행후 워킹디렉토리 단계에 머물고 수정한 문서는 워킹디렉토리 단계에 머물고 있음
